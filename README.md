# Coordiante-System


## Download Project & Install Packages


```sh
$ git clone https://github.com/deepaksaini14896/interview-2.git

$ pip install -r requirements.txt
```


## Run Project locally & By using Pytest


```sh
$ cd interview_2

$ uvicorn sql_app.main:app --reload

$ pytest

```
